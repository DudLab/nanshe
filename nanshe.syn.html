nanshe.syn package¶

The package syn provides components necessary to generate synthetic data.

Overview¶

The package syn (short for synthetic) contains components necessary for generating synthetic data. This is primarily used for testing the algorithm. However, it could be used for benchmarking algorithms or in other cases.
