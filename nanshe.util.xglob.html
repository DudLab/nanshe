nanshe.util.xglob module¶

The module xglob extends the abilities found in Python’s glob.

Overview¶

The module xglob extends the abilities found in glob. In particular, it provides a function for iterating through several glob expressions and joining them together.

API¶

nanshe.util.xglob.expand_pathname_list(*args, **kwargs)[source]¶

Takes each pathname in those given and expands them using regex.

Parameters:*pathnames (str) – pathnames to use regex on to expand.
Returns:a list of path names (without regex)
Return type:list
