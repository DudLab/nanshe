nanshe.misc package¶

The package misc contains everything that does not really belong elsewhere.

Overview¶

The package misc (short for miscellaneous) contains random components that did not really belong anywhere else. This stuff is likely not well documented or tested. As a consequence, it is safe to assume that this stuff is not stable and may be deprecated. If does ever leave this package (and is not deleted), its API may be broken from what it was here.
