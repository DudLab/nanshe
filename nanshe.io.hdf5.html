nanshe.io.hdf5 package¶

The package hdf5 provides support for IO operations on HDF5 files.

Overview¶

The package hdf5 contains a variety helper functions and methods to address IO operations on HDF5 files. Using this package requires a working copy of h5py on your PYTHONPATH.
